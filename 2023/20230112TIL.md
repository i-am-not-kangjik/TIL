## 파이썬

- random 라이브러리
    - random.randint(시작, 끝)
- set 자료형
    - 집합이다.
    - 중복이 없다.
    - set()
    - 세트 자료형도 리스트와 마찬가지로 comprehension 가능
    - 세트는 add()를 통해 값 추가
    - remove()를 통해 삭제
    - discard()로도 삭제가 가능한데, discard()는 삭제하려는 요소가 세트 자료형 내에 존재하지 않아도 KeyError를 발생시키지 않음
    - pop()도 사용이 가능하지만, 임의로 삭제해서 뭐가 삭제될지 모른다..
    - clear()로 전체삭제
    - len()으로 크기 구하기도 가능
- 함수
    - docs string
    - 정의절
    - 인수와 인자
    - 파라미터는 함수 정의절에서 사용하는 이름
    - 아규먼트는 함수 호출할 때 함수에 전달하는 실제 값
