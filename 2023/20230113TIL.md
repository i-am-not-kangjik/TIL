    - 짝수 홀수 판별
    - 평년 윤년 판별

- 람다식
    
    ```python
    lambda [parameters] : expression
    ```
    
- 조건 표현식
    
    ```python
    True일때 값 if 조건식 else False일때 값
    ```
    
- 람다와 map()을 같이 활용가능
    
    ```python
    map(lambda parameter1, parameter2, ...: expression,
    iterable1, iterable2, ...)
    ```
    
- 람다와 filter()
    
    ```python
    # 속도와 가독성 모두 리스트 컴프리헨션이 더 낫다
    map(lambda parameter: expression, iterable)
    ```
